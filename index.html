What it does

A full assistant, kept honest

Chat with any local server

Ollama, LM Studio, llama.cpp, Jan, vLLM, LocalAI — anything OpenAI-compatible. Switch models mid-conversation.

Read the page — honestly

Ask about the current page with visible token budgeting. Long pages are condensed in parts, never silently truncated.

Selection toolbar

Highlight text to explain, summarize, translate — or rewrite it in place, right inside the field you're typing in.

Chat with PDFs

Attach a PDF and ask about it — extracted on-device, through the same private pipeline.

Prompt library

Save reusable prompts and drop them in with a “/” slash command. Edit any message and branch from there.

Yours to keep

Full local history with search and export. One-click wipe of everything. It's all on your disk.

Private by architecture

Your data has nowhere else to go

The point of local is that there's no server in the middle. Open your browser's network monitor and see for yourself — nothing leaves for anyone but the model on your own machine.

No backend, no telemetry.The developer receives nothing. There is no server to receive it.
Localhost-only permissions.At install it can only reach your own machine. Any website access is granted by you, per site.
Untrusted content is fenced.Page text is treated as data, never as instructions — a documented defense against prompt injection.
You can watch it work.Every request goes to your configured local endpoint and nowhere else — visible in any network inspector.
